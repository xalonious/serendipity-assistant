# Serendipity Discord Bot

A Discord bot built to support staff operations and community management for a Roblox roleplay group. It integrates with shared backend systems to automate workflows and keep Discord, in-game systems, and scheduling data in sync.

## Core Features
- Staff and moderation utilities
- Leveling and economy systems
- Tickets, reaction roles, and utility tools
- Shift reminders synced from the Serendipity API
- Roblox integrations, including group shout posting
